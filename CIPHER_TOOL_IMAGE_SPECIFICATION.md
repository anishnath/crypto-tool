# Cipher Tool Image Specification

## Image Details

**File Name:** `cipher-encryption-tool.png`  
**Location:** `/images/site/cipher-encryption-tool.png`  
**Purpose:** SEO meta tags, Open Graph, Twitter Cards, and structured data for CipherFunctions.jsp

## Image Specifications

### Dimensions
- **Recommended:** 1200 × 630 pixels (Open Graph standard)
- **Minimum:** 600 × 315 pixels
- **Maximum:** 1200 × 1200 pixels
- **Aspect Ratio:** 1.91:1 (landscape)

### Format
- **File Format:** PNG (with transparency) or JPEG
- **Quality:** High quality (85-95% compression for JPEG)
- **Color Space:** sRGB
- **File Size:** Under 200KB for optimal loading

### Design Requirements

#### Visual Elements
1. **Lock/Encryption Icon** (primary visual)
   - Modern, minimalist lock or encryption symbol
   - Prominent placement (center or top)
   - Size: 30-40% of image height

2. **Text Elements**
   - Title: "AES Encryption Tool"
   - Subtitle: "100+ Cipher Algorithms"
   - Website: "8gwifi.org" (subtle, bottom)

3. **Color Scheme**
   - Background: Dark gradient (blue/purple) or light gradient (white/blue)
   - Primary: Blue (#6366f1) or Green (#10b981) for security/trust
   - Accent: White or light color for text/icons
   - Modern, professional appearance

4. **Style**
   - Clean, modern design
   - Minimal text (readable at small sizes)
   - Professional appearance
   - Brand-consistent colors

#### Design Concepts

**Option 1: Security/Trust Focus**
- Large lock icon with shield background
- Blue/green gradient
- "AES-256 Encryption" text
- "100+ Algorithms" badge

**Option 2: Technology Focus**
- Encryption key or cipher symbols
- Tech-style grid or circuit pattern
- Modern, minimalist design
- Blue/purple gradient

**Option 3: Tool Interface Preview**
- Small screenshot or mockup of the tool interface
- Overlay with "Free Online Tool" badge
- Professional, informative

### Content Guidelines

**Must Include:**
- Encryption/security visual (lock, key, cipher symbols)
- Tool name or reference
- Professional appearance

**Should Include:**
- Website branding (subtle)
- Trust indicators (if applicable)

**Avoid:**
- Too much text (won't be readable at small sizes)
- Cluttered design
- Old system branding
- Low-quality graphics

## Creation Options

### Option 1: AI Image Generator
Use prompts like:
- "Modern encryption tool icon with lock and cipher symbols, blue gradient background, minimalist design, 1200x630"
- "Professional AES encryption tool banner, dark blue gradient, security lock icon, clean modern design"

### Option 2: Design Tools
Use Figma, Canva, Photoshop, or similar:
- Create 1200×630 canvas
- Add lock/encryption icon
- Add gradient background
- Add minimal text
- Export as PNG/JPEG

### Option 3: Screenshot + Overlay
- Take screenshot of actual tool interface
- Add overlay with branding
- Crop to 1200×630
- Add subtle effects

## SEO Requirements

The image will be used in:
- Open Graph (`og:image`) - Facebook, LinkedIn sharing
- Twitter Card (`twitter:image`) - Twitter/X sharing
- JSON-LD schemas:
  - WebApplication `screenshot`
  - WebPage `primaryImageOfPage`

**Best Practices:**
- File name should be descriptive: `cipher-encryption-tool.png`
- Alt text equivalent should be: "AES Encryption Tool - Free Online Cipher Tool"
- Image should load quickly (optimize file size)
- Should work on both light and dark backgrounds

## File Location

Once created, save the image to:
```
src/main/webapp/images/site/cipher-encryption-tool.png
```

The SEO component will automatically reference it as:
```
https://8gwifi.org/images/site/cipher-encryption-tool.png
```

## Testing

After adding the image:
1. Validate in Google Rich Results Test
2. Test Open Graph with Facebook Debugger: https://developers.facebook.com/tools/debug/
3. Test Twitter Card with: https://cards-dev.twitter.com/validator
4. Check image loads correctly on page
5. Verify file size is reasonable (< 200KB)
